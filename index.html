How to Use This Tool

The purpose of this tool is to generate story ideas! The generated story will be split into its respective plot points, and each sentence will be generated by creating a sentence graph inspired by sentence diagramming. Adjusting the values above will change the look and feel of the generated story.

Plot Point Lengths

Plot point lengths are values from 0.0 to 1.0 that determine how much of the story (how many sentences) are dedicated to a particular plot point. These values should add up to 1.0. If not, these values will be scaled up/down such that they add up to 1.0.

Descriptiveness

This value determines the likelihood of adjectives and adverbs appearing throughout the story (while still following grammar rules).

Lung Capacity

This value determines the likelihood of conjunctions and prepositions appearing throughout the story (while still following grammar rules).

Story Length

This value determines the length (number of sentences) of the story.

Genre

Genre will significantly change the tone of the story by restricting word choice to genre-specific language.

Genre Intensity

This value affects how strictly the story draws from the genre-specific word pool. Values less than 1.0 (the maximum value) will sometimes fall back to the neutral language set.
